# mingjing

> 此项目由 learning-os BMAD 框架孵化，在 `apps/mingjing/` 下独立运行。

## 工作规则

- **独立仓库**：拥有独立 git 历史，与 learning-os 主仓库完全隔离
- **独立会话**：在此目录单独开启 Claude Code，避免加载 learning-os 的 BMAD 规则
- **部署方式**：由 learning-os 的 `scripts/deploy-app.sh mingjing` 统一推送 GitHub

## 技术规范

遵循全局 ~/.claude/CLAUDE.md 的技术选型和代码风格要求。
